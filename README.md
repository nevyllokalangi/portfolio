# Portfolio

This is a web-based portfolio project.

## Project Structure
- PHP-based application
- Organized into `app/`, `public/`, and other standard directories

## Branch Workflow
- **Development**: Main development branch
- **Staging**: Pre-production testing branch
- **Production**: Live, stable branch

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/nevyllokalangi/portfolio.git
   ```
2. Set up your web server to serve the `public/` directory.
3. Configure your database in `app/config/database.php`.
4. Install any required PHP extensions (see project code for details).

## Usage
- Access the site via your local web server (e.g., http://localhost/portfolio)
- Admin and user views are available in `app/views/`

## Contributing
- Create a feature branch from `Development`
- Open a pull request to `Development` for review
